Dental Annotation Software - User Guide

Welcome to the Dental Annotation Software! This software assists you in accurately and efficiently annotating teeth. Follow the steps below to complete your annotation process smoothly:

1. Upload Files or Folders

After opening the software, use the menu bar at the top to upload either a single file or an entire folder. If you upload a folder, a file list will appear on the left side, allowing you to select the files you want to annotate.

2. Annotate Teeth

In the Annotation Panel on the right side, select the annotation type and begin annotating teeth. Here, you can add or delete tooth annotations and select specific teeth for annotation.

3. Toolbar Features

The toolbar on the left provides several tools to support your annotations:

  • You can choose the point, line, or area tools to flexibly annotate different parts of a tooth.
  • The eraser allows you to adjust annotations, and the drag tool lets you change the view angle for better observation.

4. File Management and Saving

After completing all annotations, close the software, and a prompt will appear asking if you want to save your current annotations. Choose to save or discard them as needed to finish the annotation process.

We hope this guide helps you make the most of the Dental Annotation Software. Enjoy your experience!
